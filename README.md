## Tech Stack

### Backend

|                        |            |
| :--------------------- | ---------- |
| _Run Time Environment_ | Node.js    |
| _Server_               | Express.js |
| _Language_             | TypeScript |
| _Database_             | Postgres   |
| _ORM_                  | Prisma     |
| _Middleware_           | Zod        |
| _Password Encryption_  | argon2     |
| _User authentication_  | JWT token  |

### Frontend

|                        |                  |
| :--------------------- | ---------------- |
| _Run Time Environment_ | Node.js          |
| _Framework_            | React.js         |
| _Language_             | TypeScript       |
| _Database_             | Postgres         |
| _Routing_              | react-router-dom |
| _Middleware_           | Zod              |
